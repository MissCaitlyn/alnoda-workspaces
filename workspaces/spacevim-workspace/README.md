- Install [neovim](https://github.com/neovim/neovim)

```
sudo apt-get install -y neovim
```

- Install Spacevim 

```
curl -sLf https://spacevim.org/install.sh | bash
```

Enable +python3 and +python support 

```
pip install --user pynvim
```
